# Kunal Chauhan | Software Engineer Portfolio

A premium, interactive portfolio website built with React, Vite, and GSAP. This project features a futuristic "Spider-Man" themed identity reveal, professional experience timeline, and a selective dossier of high-impact software engineering missions.

## 🕸️ The Cinematic Identity Reveal

This portfolio utilizes custom Three.js shaders and CSS masks to create a "Secret Identity" interaction.
- **Default State**: Surfaces the Spider-Man suit as the primary base layer.
- **Hover Reveal**: A smooth, spotlight-driven mask peels back the suit to reveal the actual human portrait underneath.

## 🚀 Technical Stack

- **Front-end**: React 19, Vite, Tailwind CSS (v4)
- **Animations**: GSAP (GreenSock Animation Platform) + ScrollTrigger
- **Visuals**: Three.js (Shaders/WebGL)
- **Icons**: Custom SVGs

## 📂 Project Structure

- `src/components/Hero.jsx`: High-performance 3D shader hero with identity reveal.
- `src/components/About.jsx`: Core professional story & academic background.
- `src/components/Skills.jsx`: Categorized technical arsenal grid.
- `src/components/Experience.jsx`: Fully animated career timeline.
- `src/components/Achievements.jsx`: Achievement recognition grid.
- `src/components/Projects.jsx`: Interactive mission dossier.
- `src/components/Navbar.jsx`: Responsive navigation with Instagram profile preview.

## 🛠️ Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run in development mode**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```
